# LogicCore – Digital Logic Design Project Report

## Overview

LogicCore is a Verilog-based digital logic design project developed to strengthen practical understanding of combinational and sequential circuit design concepts. The project consists of four independent modules, each verified using dedicated testbenches.

## Modules Implemented

### 1. Four-Bit Adder

A combinational logic circuit that performs addition of two 4-bit binary numbers along with an optional carry input. The module generates a 4-bit sum and carry output.

### 2. Traffic Light FSM

A finite state machine that models a traffic light controller. The system transitions through RED, GREEN, and YELLOW states using clock-driven sequential logic.

### 3. Four-Bit ALU

A simple Arithmetic Logic Unit capable of performing:

* Addition
* Subtraction
* AND
* OR
* XOR
* NOT
* Left Shift
* Right Shift

The module also generates Zero and Carry status flags.

### 4. UART Transmitter

A UART transmitter module designed to serially transmit 8-bit data frames using start and stop bits. This module demonstrates basic serial communication concepts commonly used in embedded systems.
